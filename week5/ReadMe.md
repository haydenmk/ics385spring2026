# f2cCalc (ICS 385 Week 5 / Assignment 5b)

## Overview
This Node.js program creates a web server application converting fahrenheit to celsius based on user input.

## Packages Used
- express
- body-parser
  
## How to Run
Enter the following into the terminal in the week5/f2c folder:
- npm install express body-parser
- node index.js

## Changes Made
- Replaced previous BMI calculator functions
- Used rounding function with "Math.round()" to round integer from result
