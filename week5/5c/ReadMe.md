# Volume Calculator (ICS 385 Week 5 / Assignment 5c)

## Overview
This Node.js program creates a web server application calculating cylindrical volume based on user input of height and radius.

## Packages Used
- express
- body-parser
  
## How to Run
Enter the following into the terminal in the week5/f2c folder:
- npm install express body-parser
- node index.js

## Changes Made
- Replaced previous BMI calculator functions
- Used toFixed function with "volume.toFixed(2)" to round the decimal two places to the right
