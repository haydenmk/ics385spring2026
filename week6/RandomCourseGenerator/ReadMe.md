# Volume Calculator (ICS 385 Week 6 / Assignment 6b)

## Overview
This Node.js web server app generates a random course by combining a random array list of course IDs and course names.

## Packages Used
- express
- body-parser
- ejs
  
## How to Run
Enter the following into the terminal in the week5/f2c folder:
- npm install express body-parser
- node solution.js

## Changes Made
- Replaced the original band generator adjectives and noun arrays with course ID and course names.
- Adjusted variables and functions to match the adjusted course ID and course name changes.
