# HW15-A Google OAuth 2.0

## Author
Hayden Suzuki

## Overview
This assignment demonstrates Google OAuth 2.0 authentication using Express, Passport.js, passport-google-oauth20, MongoDB, sessions, and EJS. ChatGPT was used for troubleshooting encountered errors.

## Project Details
- Assignment: HW15-A Google OAuth 2.0
- Authentication: Google OAuth 2.0 with Passport.js
- Platform: VS Code

## Reflection
Google OAuth was very helpful for simplifying the login process. The app did not require the user to create an account entirely from scratch, and makes the process very easy, as most individuals will already have a Google account. 
The setup process was a little more complicated than the previous Passport.js assignments that we've done, but makes the User process much simpler. There is a lot more complex processes, such as the Google Client ID and the Google API setup, but
overall the process was enjoyable and is something that I could see myself using for my Term Project website, since it is centered around families and would be a lot easier for the average person with little tech skills to use.
