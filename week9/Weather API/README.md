## How to Run
1. Open project folder in VS Code
2. Enter the "weather.js" directory
3. Enter "node weather.js" into terminal line

## Purpose/Function of Code
- Creates an API that displays various information such as humidity, wind speed, etc. of Kahului (Or any location based on latitude and longitude)
