# hero-hw4 (ICS 385 Week 4 / Assignment 4a)

## Overview
This Node.js program generates and saves:
- A random superhero's name
- A random supervillain's name
- An inspirational quote
- A popular movie quote
- A web developer's famous last words

## Packages Used
- superheroes
- supervillains
- inspirational-quotes
- popular-movie-quotes
- famous-last-words
  
## How to Run
Enter the following into the terminal:
- npm install
- node index.js

## Output Files
- superhero.txt
- supervillain.txt
- inspirational-quote.txt
- popular-movie-quote.txt
- famous-last-words.txt

- Added a function using the File System module to create .txt files of all terminal outputs.
